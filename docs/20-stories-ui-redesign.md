# Stories UI Redesign

This phase upgrades the stories section from a simple catalog into an immersive reading experience.

## What changed
- richer hero with clear entry points
- stronger category cards with preview and progress
- dedicated reading card for the active story
- next/previous navigation inside the reader
- related stories list inside the same category
- improved bookmark/favorite/share/copy controls
- calmer dark-mode and reduced-motion-friendly polish

## Design direction
- immersive moral reading
- clear hierarchy
- low-noise motion
- strong readability over decoration
