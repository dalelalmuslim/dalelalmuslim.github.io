# Azkar Reminder Integration

## الهدف
ربط تفضيلات التذكير داخل قسم الأذكار بخدمة الإشعارات الحالية بشكل ثنائي الاتجاه.

## ما الذي تغيّر
- تغيير reminder mode من قسم الأذكار يزامن Notification settings تلقائيًا.
- حفظ Notification settings من صفحة الإعدادات يحدّث reminder state داخل قسم الأذكار.
- إضافة reminder type جديد: `prayer` لأذكار ما بعد الصلاة.
- إضافة صف إعدادات مستقل داخل صفحة الإعدادات لأذكار ما بعد الصلاة.

## قرار الدمج
- `off` -> تعطيل كل تنبيهات الأذكار.
- `morning` -> تفعيل تنبيه الصباح فقط.
- `evening` -> تفعيل تنبيه المساء فقط.
- `prayer` -> تفعيل تنبيه ما بعد الصلاة فقط.
- `sleep` -> تفعيل تنبيه النوم فقط.
- `smart` -> تفعيل الصباح + المساء + النوم.

## ملاحظات
- الربط يعتمد على Browser Notifications الحالية، وليس على background push خارجي.
- إذا كانت الإشعارات محظورة، تبقى تفضيلات الأذكار محفوظة لكن التنبيه النظامي لن يعمل حتى يسمح المستخدم بالإشعارات.
