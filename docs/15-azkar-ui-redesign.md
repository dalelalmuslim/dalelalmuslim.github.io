# Azkar UI Redesign

## What changed
- stronger azkar hero and resume surface
- filter chips for daily / period-based views
- richer category cards with progress and state labels
- session panel with progress stats and quick preference toggles
- calmer, more premium visual language for azkar item cards
- focus mode and large-text affordances exposed inside the section UI

## Why
The azkar section should feel like a daily ritual surface, not a plain content list. The redesign reduces friction, improves scanning, and makes return usage easier.

## Notes
- progress identity remains stable by `slug`
- adding a new azkar category is still manifest-driven
- focus mode only changes azkar presentation; it does not affect other sections
